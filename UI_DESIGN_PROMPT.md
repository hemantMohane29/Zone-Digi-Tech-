## COLOR SCHEME & STYLING

**Typography:**
- Font family: "Dosis" for display (headers), "Inter" or "Poppins" for body
- Font weights: Bold for headlines, Regular for body, Semibold for CTAs
- Letter spacing: Wide tracking on uppercase labels and tags

**Button Styles:**

1. **Primary Button (Orange Gradient)**
   - Background: linear-gradient(135deg, #e07b00, #f9b84a)
   - Color: white
   - Padding: px-6 py-3
   - Rounded: rounded-full
   - Hover: Shift colors slightly, add shadow

2. **Secondary Button (Outline)**
   - Border: 2px stone-800/stone-200
   - Color: stone-800/stone-200
   - Padding: px-6 py-3
   - Rounded: rounded-full
   - Hover: Background fill with orange color

3. **Icon Buttons (Social/Navigation)**
   - Background: stone-800
   - Size: w-8 h-8
   - Rounded: rounded-lg
   - Hover: bg-saffron-600, text-white

**Card Styles:**
- Rounded corners (rounded-2xl)
- Shadow on hover (shadow-lg, shadow-xl)
- Smooth transitions (duration-300, duration-200)
- Hover lift effect (-translate-y-1)
- Border subtle (border-stone-200 / dark border-stone-800)

**Layout:**
- Max width container: max-w-7xl
- Padding: px-4 sm:px-6 lg:px-8
- Sections: py-12 md:py-20 lg:py-24
- Double column layout for larger content
- Responsive grid: grid-cols-1 md:grid-cols-2 lg:grid-cols-3 lg:grid-cols-4

## RESPONSIVE DESIGN

- Mobile-first approach
- Breakpoints: sm, md, lg
- Hamburger menu on mobile
- Stack layout on mobile (single column)
- Two-column on tablet
- Three-column on desktop
- Touch-friendly buttons and interactive elements
- Readable font sizes on all devices

## DARK MODE

- Complete dark mode support using Tailwind's dark: class
- Toggle switch in navbar to switch themes
- Persist theme preference (localStorage)
- Smooth color transitions when toggling
- Adjusted colors for readability in dark mode

- Built with React 18.3.1 + TypeScript
- Styled with Tailwind CSS 3.4.1
- Icons from lucide-react
- Vite 5.4.2 as build tool
- Theme context for light/dark mode management
- Custom hook for scroll animations
- Multi-page routing (no external router library, state-based)
- Responsive and mobile-optimized
